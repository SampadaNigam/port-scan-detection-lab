# 🛡️ Port Scan Detection Lab

A Splunk-based detection lab demonstrating how to identify port-scanning activity using UFW firewall logs and Nmap scans.  
All detection logic, dashboards, alerts, and test commands are documented in separate `.md` files.

## 📁 Repository Structure
- `detections/` — SPL queries and detection logic  
- `dashboards/` — dashboard XML + screenshots  
- `alerts/` — alert configuration and logic  
- `nmap-commands/` — Nmap commands used to generate logs  
- `images/` — diagrams and screenshots
